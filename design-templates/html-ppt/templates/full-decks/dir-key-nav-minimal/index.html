The Fix

The answer is not
a larger window.

Write your knowledge, preferences, and history into a file system.
Let the LLM read that system before every conversation.

× Context stuffing

Put everything in the prompt: expensive, slow, and eventually full.

✓ Files + retrieval

Load on demand, never overflow, and keep the structure diffable.

Minimal Setup

Four lines of YAML
are enough to start.

memory:
  root: ~/.llm-wiki
  format: markdown
  retrieval: hybrid  # embedding + bm25

You now have a second brain that grows over time. Every conversation reads and updates it.
