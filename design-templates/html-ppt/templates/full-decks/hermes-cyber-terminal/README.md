# hermes-cyber-terminal

Black background (`#0a0c10`) with terminal chrome, scan lines, mint-glow type, and a JetBrains Mono terminal feel. It combines dark code boxes with an evidence-led comparison into an honest cyber review.

**Visual traits:** A 56px cyber grid, CRT vignette, translucent scanline overlay, traffic-light window chrome, `$ prompt` command-line headings, mint `#7ed3a4` text-shadow glow, global monospace type, stroke-only bar charts, a blinking cursor, and amber/green/red severity tags.

**Use when:** Reviewing a developer tool, CLI, or agent; showing benchmarks, traces, or diffs; or presenting long traces and code in an honest technical-review voice.

**Source inspiration:** Hermes terminal review decks.

**Path:** `templates/full-decks/hermes-cyber-terminal/index.html`
