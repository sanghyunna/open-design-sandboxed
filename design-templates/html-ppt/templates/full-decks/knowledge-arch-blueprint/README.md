# knowledge-arch-blueprint

Cream paper (`#F0EAE0`), rust red (`#B5392A`) as a single accent, hard black-bordered cards, and dashed feedback-loop arrows. The result feels like a technical-whitepaper architecture diagram or building blueprint.

**Visual traits:** Warm paper, a faint 48px blueprint grid, firm 2px black card borders, a horizontal pipeline of step boxes with a raised hero box, a rust insight callout, uppercase kickers with 2.5–4px tracking, dashed SVG feedback arrows, oversized Playfair serif metrics, and no gradients or shadows.

**Use when:** Explaining system architecture, data flow, or a process breakdown; when the work should read as a serious technical whitepaper rather than a marketing post; or when the result should be easy to screenshot into a README.

**Source inspiration:** a knowledge-architecture blueprint deck.
