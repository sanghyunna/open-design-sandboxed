Problem · Solution

Original vs. upgraded

Original Karpathy

Write once

Capture → transcribe → archive. Errors remain errors. Without a loop or correction mechanism, more notes mean more confusion.

UPGRADE V2

Feedback loop

When AI uses the knowledge base, record every miss, hallucination, and stale fact; feed it back to source files so notes can self-correct.

Implementation · Skill Manifest

How to build the feedback loop

A 100-line agent skill that writes whether AI worked well back to the vault as individual revision records.

# skills/wiki-feedback/SKILL.md
name: wiki-feedback
trigger: "after every retrieval"

on_hit:     record(query, path, used=true)
on_miss:    record(query, reason="not-in-vault")
on_wrong:   record(query, correction, path)

nightly:
  - aggregate misses → suggest new notes
  - aggregate wrongs → diff-patch old notes
  - commit to git, open PR for human review
