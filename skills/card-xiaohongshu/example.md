# Five Habits That Make Your AI Editor Easier to Use

## Cover
Save this · Essential for AI-tool users

## Card 1: Stop Chasing the Perfect Prompt
No prompt is perfect on the first try. Treat it as a **conversation**, not SQL.

## Card 2: Let AI See Your Format
Before pasting Markdown or CSV, say: "I want the output to look like X."
One concrete reference beats fifty adjectives.

## Card 3: Use Templates Instead of Rewriting
Save useful prompts as templates. Choose one directly in HTML Anything;
nine world-class design systems are ready to use.

## Card 4: Send Diffs for Revisions
Regeneration does not start from zero. Send only the diff so AI adjusts the existing HTML,
saving tokens while preserving the design.

## Card 5: Reuse Your Local Session
Run local Claude, Cursor, or Codex without an API key. There is no marginal API cost,
and it can be faster than a remote API.

## Summary
More tools are not always better; a smooth workflow is.
Follow for practical AI-workflow updates every week.
