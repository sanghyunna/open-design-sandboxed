# New-User Onboarding · HTML Anything

> Whiteboard sticky-note flowchart, six nodes, one branch

## Flow
1. **Open localhost:3456** — The browser enters the main interface.
2. **Detect local agents** — Scan PATH and list signed-in CLIs.
3. **Choose a template** — 75 templates are grouped by scenario.
4. **Paste content or upload a file** — Markdown, CSV, and Excel are supported.
5. **Press ⌘+Enter to convert** — An agent streams through SSE and refreshes the preview live.
6. **Copy or download in one step** — Export as share-ready text, PNG, or HTML.

Branch: if no local agent is found, guide installation of Claude, Cursor, or Codex.
